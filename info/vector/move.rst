Element Reordering
==================

.. rubric:: Implicit Interface
.. list-table::
   :widths: auto
   :width: 100%

   * - `vector_swap()`
     - Swap the element at index *i* with the element at index *j* in the *vector*
   * - `vector_move()`
     - Move the element at index *source* to index *target* in the *vector*

.. rubric:: Explicit Interface
.. list-table::
   :widths: auto
   :width: 100%

   * - `vector_swap_z()`
     - Swap the element at index *i* with the element at index *j* in the *vector*
   * - `vector_move_z()`
     - Move the element at index *source* to index *target* in the *vector*
